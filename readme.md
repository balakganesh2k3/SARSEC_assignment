# SASRec on MovieLens-1M
PyTorch implementation of SASRec trained on MovieLens-1M. Given a user's watch history, predicts what they'll interact with next.

---
## Files
```
src/
├── train.py # training loop, early stopping, config logging
├── evaluate.py # full-ranking eval (NDCG@K, Recall@K)
├── model.py # SASRec architecture
├── module.py # attention and feedforward blocks
├── dataset.py  # dataset class, per-position negative sampling
├── sampler.py # negative item sampler
├── data_preprocessing.py # ratings.dat → data.pkl
├── interface.py # shared constants (MAX_LENGTH, TOKEN_PADDING)
└── plot.py # training curve from log file
data/
├── ratings.dat # raw MovieLens-1M (download separately)
└── processed/
    └── data.pkl # generated by data_preprocessing.py
checkpoints/
└── best_model.pt # saved during training
```
---
## Setup
Python 3.11 or 3.12. PyTorch doesn't support 3.13+ yet.
```bash
pip install torch --index-url https://download.pytorch.org/whl/cu121
pip install numpy pandas tqdm matplotlib
```
Download MovieLens-1M from [grouplens.org/datasets/movielens/1m](https://grouplens.org/datasets/movielens/1m/) and put `ratings.dat` in `data/`.

## Running
**Step 1 — preprocess:**
```bash
cd src
python data_preprocessing.py
```
Filters ratings ≥ 4, builds chronological sequences, drops users with fewer than 5 interactions, writes `data/processed/data.pkl`. Takes about 10 seconds.

**Step 2 — train:**
```bash
python train.py
```
Prints a config block, then one line per epoch. Saves the best checkpoint to `checkpoints/best_model.pt` and logs everything to `Training_on_cuda.txt`. Test results print at the end.

**Step 3 — plot:**
```bash
python plot.py
```
Reads `Training_on_cuda.txt`, saves training curve to `../results/sasrec_line.png`.

## Hyperparameters

| Parameter | Default |
|---|---|
| Hidden size | 50 |
| Attention heads | 1 |
| Transformer blocks | 2 |
| Max sequence length | 200 |
| Dropout | 0.2 |
| Batch size | 512 |
| Learning rate | 0.001 |
| Optimizer | Adam (β1=0.9, β2=0.98) |
| Scheduler | Linear warmup (10 ep) + CosineAnnealingLR |
| Early stopping patience | 5 |

To try a different config, change the `SASRec(...)` call in `train.py`:

```python
model = SASRec(num_items=data["num_items"], hidden_size=100, head_nums=2, block_nums=2, maxlen=MAX_LENGTH dropout_rate=0.2)
```
